# Grille d'evaluation academique GitHub

## Bareme
| Critere | Description | Points |
|---|---|---:|
| Convention de nommage | Format `Esprit-[PI]-[Classe]-[AU]-[NomDuProjet]` respecte | 0.5 |
| Description du projet | Description publique claire avec branding institutionnel | 0.5 |
| Topics conformes | Topics obligatoires correctement renseignes | 0.5 |
| Mots-cles README | Presence de `Esprit School of Engineering` et sections obligatoires | 0.25 |
| Hebergement (facultatif) | Projet deploye et lien fonctionnel | 0.25 |

## Regle d'ajustement
- Si hebergement non fourni, le critere README peut monter a 0.5 selon consigne PI.

## Exemple de calcul
| Critere | Note obtenue |
|---|---:|
| Convention de nommage | 0.5 |
| Description | 0.5 |
| Topics | 0.25 |
| README | 0.5 |
| Hebergement | 0 |
| **Total** | **1.75 / 2.0** |

## Commentaires qualitatifs recommandes
- Qualite des commits
- Clarte de la documentation
- Qualite technique globale
- Qualite de l'organisation du repository
