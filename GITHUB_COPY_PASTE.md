## 🔗 Commande Git pour Mettre à Jour le Remote

Après avoir renommé le dépôt sur GitHub, exécutez cette commande localement:

```bash
git remote set-url origin https://github.com/your-username/Esprit-PIDEV-3A-2026-Wordly-ELearning.git
```

⚠️ **Remplacez `your-username` par votre nom d'utilisateur GitHub!**
