┌─ STEP 1: START SERVICES ──────────────────────────────────────────────────────┐
│                                                                               │
│ Terminal 1:  cd AbonnementService && mvn spring-boot:run                     │
│ Terminal 2:  cd UserService && mvn spring-boot:run                           │
│ Terminal 3:  cd ApiGateway && mvn spring-boot:run (optional)                 │
│                                                                               │
│ Wait for all services to start successfully                                  │
└───────────────────────────────────────────────────────────────────────────────┘

┌─ STEP 2: CREATE TEST DATA ────────────────────────────────────────────────────┐
│                                                                               │
│ Option A (Recommended):                                                       │
│   .\CREATE_TEST_REMINDER_DATA.ps1                                             │
│                                                                               │
│ Option B (Manual):                                                            │
│   mysql -u root -p -P 3307 < CREATE_TEST_REMINDER_DATA.sql                   │
│   (Edit file first to set user ID and subscription ID)                       │
│                                                                               │
│ This creates 5 test payments with different expiration dates                 │
└───────────────────────────────────────────────────────────────────────────────┘

┌─ STEP 3: TEST THE SYSTEM ─────────────────────────────────────────────────────┐
│                                                                               │
│ Option A (Recommended):                                                       │
│   .\TEST_REMINDER_SYSTEM.ps1                                                  │
│                                                                               │
│ Option B (Manual API calls):                                                  │
│   1. curl -X POST http://localhost:8083/api/subscription-reminders/check-now │
│   2. curl http://localhost:8083/api/subscription-reminders/user/1            │
│   3. curl http://localhost:8083/api/subscription-reminders/all               │
│                                                                               │
└───────────────────────────────────────────────────────────────────────────────┘

┌─ STEP 4: TEST FRONTEND ───────────────────────────────────────────────────────┐
│                                                                               │
│ Terminal 4:  cd frontend/angular-app && ng serve --port 4200                 │
│                                                                               │
│ Then:                                                                         │
│   1. Open http://localhost:4200                                              │
│   2. Login with your account                                                 │
│   3. Look for bell icon 🔔 in top right header                               │
│   4. Click bell to see reminders dropdown                                    │
│   5. Check colors: 🔴 Red (expired), 🟠 Orange (today), 🟡 Yellow (soon)    │
│                                                                               │
└───────────────────────────────────────────────────────────────────────────────┘

┌─ TROUBLESHOOTING ─────────────────────────────────────────────────────────────┐
│                                                                               │
│ No reminders showing?                                                         │
│   → Run: .\TEST_REMINDER_SYSTEM.ps1                                           │
│   → Check: mysql -u root -p -P 3307 -D elearning_db                          │
│            -e "SELECT * FROM payments WHERE statut = 'Validé';"              │
│                                                                               │
│ Bell icon not visible?                                                        │
│   → Make sure you're logged in                                               │
│   → Check browser console (F12) for errors                                   │
│   → Clear Angular cache: Remove-Item -Recurse -Force .angular                │
│                                                                               │
│ Services not starting?                                                        │
│   → Check ports: 8083 (Abonnement), 8085 (User), 8888 (Gateway)             │
│   → Check MySQL is running on port 3307                                      │
│   → Check Java version: java -version (need Java 17+)                        │
│                                                                               │
└───────────────────────────────────────────────────────────────────────────────┘
